# Возможности
- **Список фильмов текущего года с высоким рейтингом**.
- **Добавление фильма в избранные**.
- **Просмотр деталей фильма**.
- **Просмотр деталей актера**.

# Стек технологий

- **Kotlin**.
- **Compose**.
- **Многомодульность**.
- **Single Activity**.
- **MVVM (ViewModel + StateFlow)**.
- **Hilt (DI)**.
- **Retrofit**.
- **Room**.
- **JUnit и Mockito**.
